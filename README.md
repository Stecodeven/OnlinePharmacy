#针对项目成员的git 使用：

本地项目文件目录比如为  main/
打开命令行，cd到main目录下，
	git init回车

然后克隆本项目  
	git clone git@gitee.com:Callmejp/OnlinePharmacy.git

此时远程仓库的项目所有文件都已经克隆到本地了。
项目成员可以根据情况作出任意修改

git remote -v  查看本地关联的远程仓库
